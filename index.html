How the map works

The map uses your property-map.png image as the background and places clue markers using percentage coordinates from config.js.
  • Replace property-map.png with your final clean map image.
  • Edit each clue’s zone.x and zone.y values in config.js.
  • Values are percentages of the image width and height.
  • Refresh the page after each change to check placement.
